Welcome to the Flashphoto/MIA Homepage

FlashPhoto

FlashPhoto is an image editor that offers many tools, such as Pen, Spray Can, Highlighter, Stamp, Blur, and Rainbow Pen, and filters, such as MotionBlur, Sharpen, Saturate, Quantize, and many more! These tools and filters can be used to create original images or edit your existing images. The possibilities are endless!

MIA: Medical Imaging and Analysis

MIA is a special-purpose image editor. MIA has been designed specifically for users in the medical profession. MIA offers a selection of essential tools and filters necessary for medical imaging. MIA also offers the ability to load a stack of images.

Both of the above applications can be run in graphical mode and command-line mode. See Converting and Segmenting a CT Scan Using MIA for a sample tutorial using each of these modes.
