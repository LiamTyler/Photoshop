Converting and Segmenting a CT Scan Using MIA

Converting and Segmenting a CT Scan Using the MIA Application

  1. To save your segmented scans in a new directory, you must first create the directory. To create the directory, run the following command in your terminal while in the location where you want the new directory. 
    2. mkdir <directory name>
  2. Open the MIA application by running the following command in your terminal from the main project directory. 
    3. ./build/bin/MIA
  3. Find the first image you want to segment in the Image I/O panel on the right side of the Controls window.
  4. Double click on the image. The name of the image should be in the "Image:" box.
  5. Click on Load Canvas. You should see your scan in the MIA window.
  6. To reduce noise, Blur the image by changing the Blur Amount to 3.0 and clicking on the "Blur" button.
  7. To segment the image, change the number of Bins to 4 and click on the "Quantize" button.
  8. Now, find the directory you created in the Image I/O panel and double click on it.
  9. In the "Image:" box, enter a new name for your segmented image.
  10. Click on Save Canvas. You should see your new image in the Image I/O panel.
  11. Find the next image you want to segment in the Image I/O panel on the right side of the Controls window, and repeat steps 4-10 to segment the next image.
