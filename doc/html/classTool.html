Detailed Description

Tool is the representation of a particular BrushWork tool. This baseclass is abstract and does not provide a default mask. Tools inherited from this class MUST:
(1) Provide a name
(2) Define the mask. You will get an assert error if you do not.
Inherited classes may optionally override the default color_blend_math().

Default implementation for applying the tool to the PixelBuffer.

Uses the current_color and last position of the tool being applied (last_x and last_y) to determine how to apply the mask to the buffer.

Parameters
buffThe buffer of pixels to apply the tool to
[in]current_colorThe current color of the tool
[in]xThe x position of where to apply the tool to
[in]yThe y position of where to apply the tool to
[in]last_xThe x position of the last time the tool was applied (only applicable while the mouse is being dragged)
[in]last_yThe y position of the last time the tool was applied (only applicable while the mouse is being dragged)
